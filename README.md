# Spring-Rent-Storage
Spring web app aimed to rent or share storage space.  
Application's users will have ability to publish their own storage space which will be available to be seen by clients.

## Technologies
- Spring
- MySQL database
- Hibernate ORM
- Thymeleaf
- Slf4j
- Maven
- Tomcat maven plugin
- Lombok
- Google Guava
- Bootstrap 3
